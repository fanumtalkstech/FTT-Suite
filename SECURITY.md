## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by emailing:

📧 **fanumtax.westsidebeamrp@gmail.com**

Please include as much detail as possible, such as:
- A description of the vulnerability
- Steps to reproduce the issue
- The version of the tool affected
- Any relevant logs, screenshots, or generated scripts (with sensitive info removed)

**Please do not open a public GitHub issue for security vulnerabilities.** Reporting privately via email helps prevent the issue from being exploited before a fix is available.

## What to Expect

- You should receive an acknowledgment of your report within a few days.
- If the vulnerability is confirmed, a fix will be prioritized and released as soon as possible, and you'll be credited (unless you prefer to remain anonymous).
- If the report is declined (e.g., not reproducible, out of scope, or a false positive), you'll receive an explanation of why.

Thank you for helping keep this project and its users safe.
