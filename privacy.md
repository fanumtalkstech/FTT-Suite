# Privacy Policy

**FTT Suite** does not collect, store, transmit, or share any user data whatsoever.

## Data Collection

- No telemetry, analytics, or tracking is built into this tool.
- No data is sent to the author or any third party.
- No usage statistics, crash reports, or logs leave your machine.

## What This Tool Does Locally

All actions performed by this tool — application installs and system tweaks — run **locally on your machine** using standard, built-in Windows tools:

- `reg` for registry edits
- `powercfg` for power plan changes
- `winget` for application installs

## Network Activity

The only network activity involved is `winget` contacting Microsoft's official package servers to download the applications you select. This is standard Windows Package Manager behavior and is not controlled, modified, or intercepted by this tool in any way.

## Summary

This tool only generates a script. It does not phone home, and the author has no visibility into how, when, or where it is used.

If you have questions about this policy, please open an issue in this repository.
